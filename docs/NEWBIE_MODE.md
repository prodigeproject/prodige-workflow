# Newbie Mode

This mode is for non-technical vibe coders.

You do not need to know:
- architecture
- skills
- workflows
- agent roles
- cache
- sessions
- locks

Just use commands.

## Minimum commands

```text
/init from idea: ...
/design ...
/build approved ...
/fix ...
/audit
/review
/sync
/status
```

## What AI should explain to you

For every important step, AI should explain:

1. What I understood.
2. What I will change.
3. Why this matters.
4. What risk exists.
5. What needs your approval.
6. What happens next.

## When AI must stop and ask

AI must stop before:
- changing architecture
- changing database schema
- replacing auth system
- deleting major code
- rewriting large modules
- changing payment logic
- changing production deployment
- making irreversible changes
